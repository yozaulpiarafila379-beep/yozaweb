Object Oriented Programming (OOP)

Object Oriented Programming (OOP) adalah paradigma pemrograman yang berfokus pada objek sebagai unit utama. Setiap objek memiliki state (data) dan behavior (perilaku). Tujuan OOP adalah membuat kode yang modular, reusable, dan mudah dipelihara dalam sistem yang kompleks.

1. Class dan Object

Contoh pendefinisian class Car dengan dua attribute (model dan color) serta dua method (start dan stop).

Contoh Kode 1: Class Car dengan Metode Start & Stop

class Car {
  String? model;
  String? color;

  void start() {
    print('Car started!');
  }

  void stop() {
    print('Car stopped!');
  }
}

void main() {
  Car myCar = Car();
  myCar.model = "Sedan";
  myCar.color = "Merah";

  myCar.start();
  myCar.stop();
}

Contoh Kode 2: Class Car dengan Menampilkan Properti

class Car {
  String model;
  String color;

  Car(this.model, this.color);

  void start() {
    print('Engine Online: $model');
  }
}

void main() {
  Car myCar = Car("Sedan", "Merah");

  myCar.start();
  print(myCar.color);
}

2. Contoh Kelas Lain dalam OOP

Object Oriented Programming (OOP) adalah paradigma pemrograman yang berorientasi pada objek. Dalam OOP, program dibangun menggunakan class dan object yang memiliki data (atribut) dan perilaku (method). Tujuan OOP adalah membuat kode lebih terstruktur, mudah dipahami, mudah dikembangkan, dan dapat digunakan kembali. Konsep utama dalam OOP meliputi Encapsulation, Inheritance, Polymorphism, dan Abstraction.

Contoh Kode 1: Class Mahasiswa

class Mahasiswa {
  String nama;
  String nim;
  double ipk;

  Mahasiswa(this.nama, this.nim, this.ipk);

  void tampilkanData() {
    print('Nama : $nama');
    print('NIM : $nim');
    print('IPK : $ipk');
  }
}

void main() {
  Mahasiswa mhs1 = Mahasiswa("Yoza Ulpia Rafila", "251410004", 3.85);
  mhs1.tampilkanData();
}

Contoh Kode 2: Class Person dengan Properti Tersembunyi

class Person {
  String _name;
  int age;

  Person(this._name, this.age);

  void setName(String name) => _name = name;

  String getName() => _name;
}

void main() {
  Person person = Person("Yoza Ulpia Rafila", 19);

  print(person.age);
  person.setName("Yoza Ulpia Rafila");
  print(person.getName());
}

Contoh Kode 3: Class Product dengan Konstruktor Sederhana

class Product {
  String nama;

  Product(this.nama);
}

void main() {
  Product produk1 = Product("Laptop Gaming");
  print(produk1.nama);
}

Contoh Kode 4: Class Produk dengan Konstruktor Bernama

laptop gaming

class Produk {
  final String id, nama;
  final double harga;
  final int stok;
  final String kategori;

  Produk({
    required this.id,
    required this.nama,
    required this.harga,
    this.stok = 0,
    this.kategori = "Umum",
  });
}

void main() {
  Produk laptop = Produk(
    id: "P001",
    nama: "Laptop Gaming",
    harga: 15000000,
    stok: 15,
    kategori: "Elektronik",
  );

  Produk smartphone = Produk(
    id: "P002",
    nama: "Smartphone Flagship",
    harga: 12000000,
    stok: 25,
  );

  Produk kamera = Produk(
    id: "P003",
    nama: "Kamera Mirrorless",
    harga: 20000000,
  );

  print(laptop.nama);
  print(smartphone.nama);
  print(kamera.nama);
}

Kesimpulan

Dari berbagai contoh kode yang telah ditampilkan, dapat disimpulkan bahwa Object Oriented Programming (OOP) memudahkan pengelolaan kode dengan mengelompokkan data dan perilaku ke dalam kelas. Setiap kelas dapat memiliki atribut untuk menyimpan data dan method untuk menjalankan aksi.

Konsep seperti konstruktor (posisional dan bernama) membantu dalam membuat objek dengan cara yang lebih fleksibel, sedangkan properti tersembunyi membantu dalam melindungi data agar tidak diakses secara sembarangan. Dengan menerapkan OOP, kode yang dibuat akan menjadi lebih terstruktur, mudah diperbaiki, dan dapat digunakan kembali untuk kebutuhan yang berbeda.
