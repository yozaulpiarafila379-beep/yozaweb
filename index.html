2. Pengertian OOP

Object Oriented Programming adalah metode pemrograman yang menggunakan konsep class dan object untuk menyusun program agar lebih terstruktur, modular, dan mudah dikembangkan.

3. Konsep Dasar OOP

  • Class – Cetakan untuk membuat object
  • Object – Instance dari class
  • Encapsulation – Menyembunyikan data
  • Inheritance – Pewarisan sifat
  • Polymorphism – Banyak bentuk
