2. Apa Itu OOP?

Object Oriented Programming adalah gaya pemrograman yang menggunakan konsep class dan object untuk menyusun logika program. Dengan OOP, kode jadi lebih teratur, modular, dan mudah dipelihara.

3. Konsep OOP yang Sering Dipakai

  • Class — Cetakan objek
  • Object — Instance class
  • Method — Aksi dalam class
  • Attribute — Data milik object

4. Contoh Program Class & Object

// Dart — class kendaraan class Vehicle { String brand; int year; Vehicle(this.brand, this.year); void info() { print("Vehicle: $brand, Year: $year"); } } void main() { var car = Vehicle("Toyota", 2020); car.info(); }

5. Inheritance (Pewarisan)

Inheritance membuat class baru dapat mewarisi sifat (properties & methods) dari class lain.

class Animal { void eat() { print("Eating"); } } class Cat extends Animal { void sound() { print("Meow"); } } void main() { Cat kitty = Cat(); kitty.eat(); kitty.sound(); }
