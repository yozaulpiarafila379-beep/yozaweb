Object Oriented Programming (OOP)

Object Oriented Programming (OOP) adalah paradigma pemrograman yang berfokus pada objek sebagai unit utama. Setiap objek memiliki state dan behavior. Tujuan OOP adalah membuat kode lebih modular dan mudah dipelihara.

1. Class dan Object

Contoh Kode 1: Class Car

class Car {
String? model;
String? color;

void start(){
print("Car started");
}

void stop(){
print("Car stopped");
}
}

void main(){
Car myCar = Car();
myCar.model = "Sedan";
myCar.color = "Merah";

myCar.start();
myCar.stop();
}

Kesimpulan

OOP membantu membuat program lebih terstruktur dengan class, object, constructor, dan encapsulation.
