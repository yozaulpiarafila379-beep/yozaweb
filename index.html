1. Deklarasi Kelas dan Variabel Instance

class Produk {
  final String id, nama; 
  final double harga; 
  final int stok;   
  final String kategori;

  // ... konstruktor dan isi kelas ...
}

class Produk - Mendefinisikan kelas bernama Produk, yang berfungsi sebagai cetakan untuk membuat objek produk dengan ciri-ciri tertentu.

final String id, nama - Variabel instance bertipe String yang bersifat final (nilainya tidak bisa diubah setelah diinisialisasi). id untuk kode produk unik, nama untuk nama produk.

final double harga - Variabel bertipe double untuk menyimpan harga produk (bisa berisi angka desimal).

final int stok - Variabel bertipe int untuk menyimpan jumlah stok produk yang tersedia.

final String kategori - Variabel bertipe String untuk mengelompokkan produk ke dalam kategori tertentu.

2. Konstruktor Kelas Produk

Produk({
  required this.id,
  required this.nama,
  required this.harga,
  this.stok = 0,
  this.kategori = 'Umum',
});

Produk({ ... }) - Konstruktor dengan parameter bernama (named parameters), memudahkan pembacaan saat membuat objek.

required this.id, required this.nama, required this.harga - Parameter yang wajib diisi saat membuat objek, karena ditandai dengan required. this. digunakan untuk menghubungkan parameter dengan variabel instance kelas.

this.stok = 0 - Parameter opsional dengan nilai default 0 (jika tidak diisi, stok akan dianggap 0).

this.kategori = 'Umum' - Parameter opsional dengan nilai default 'Umum' (jika tidak diisi, produk masuk kategori Umum).

3. Fungsi Utama (main) dan Pembuatan Objek

void main() {
  Produk laptop = Produk(
    id: "P001",
    nama: "Laptop Gaming",
    harga: 15000000,
    stok: 15,
    kategori: "Elektronik",
  );

  Produk smartphone = Produk(
    id: "P002",
    nama: "Smartphone Flagship",
    harga: 12000000,
    stok: 25,
  );

  Produk kamera = Produk(
    id: "P003",
    nama: "Kamera Mirrorless",
    harga: 20000000,
  );

  print(laptop.nama); 
  print(smartphone.nama); 
  print(kamera.nama); 
}

void main() - Fungsi utama yang akan dijalankan pertama kali saat program Dart berjalan; merupakan titik awal eksekusi program.

Produk laptop = Produk(...) - Membuat objek laptop dari kelas Produk dengan mengisi semua parameter (termasuk kategori yang diubah menjadi "Elektronik").

Produk smartphone = Produk(...) - Membuat objek smartphone; parameter kategori tidak diisi, sehingga menggunakan nilai default 'Umum'.

Produk kamera = Produk(...) - Membuat objek kamera; parameter stok dan kategori tidak diisi, sehingga menggunakan nilai default 0 dan 'Umum'.

print(laptop.nama) - Menampilkan nilai properti nama dari objek laptop ke konsol (output: "Laptop Gaming"). Demikian juga untuk smartphone (output: "Smartphone Flagship") dan kamera (output: "Kamera Mirrorless").
